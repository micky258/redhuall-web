# 📁 Carpeta de imágenes

Aquí debes colocar las imágenes de tu proyecto.

## Para el logo de RedHuall:

1. Descarga tu logo desde Google Drive (archivo `.png` o `.jpg`)
2. Renómbralo a `logo.png` (o `logo.jpg` según corresponda)
3. Arrástralo a esta carpeta (`public/images/`)
4. El logo aparecerá automáticamente en la página

## Otros archivos que puedes agregar:
- `favicon.ico` → ícono que aparece en la pestaña del navegador
- Fotos de trabajos realizados
- Fotos de productos
- Imágenes de la empresa
